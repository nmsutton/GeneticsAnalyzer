Welcome to this sample genetic experiment analysis site!
This site is for displaying some methods of presenting and interacting with genetic experiment results. For geneticists, biologists, and public consumers with interests in genetics this displays some ways to retrieve and investigate results from genetic experiments. A concept behind this work is that experiments can be run at remote computers and individuals can receive results anywhere through the web.

This site is presently still under active development and a lot will be added to it by the end of 06/05/13.

Please use the navigation bar at the to explore some of the site's features! Some future work includes providing a user friendly and internet accessible way for users to create new types of experiments with custom datasets they provide. Private secure access to uploaded data is a future feature being considered as well.
